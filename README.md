# Sales Performance Analysis Looker Studio

## Project Overview
This project develops a centralized sales dashboard for Maru Footwear Sales, a footwear retailer operating across multiple e-commerce platforms including Shopee, Tokopedia, Lazada, and Blibli.

## Objective 
- Analyze sales performance across multiple marketplaces on each platform.
- Identify top-performing products and categories based on key metrics. 
- Support data-driven decision making by providing interactive insights that help optimize sales strategy, pricing, and marketplace focus.

## ToolS
Google Looker Studio

## Dataset 
Maru Footwear Sales

## Analysis Approach
- Analyzing shoe sales data from various e-commerce platforms such as Shopee, Tokopedia, Lazada, and Blibli.
- Visualizing sales performance on e-commerce platforms in an interactive Looker Studio dashboard to support data-driven decision making.
